# GitHub Labels for EgyptianAgent Project

The following labels should be created in the GitHub repository for proper issue tracking and organization:

## Core Labels

### `core`
- **Color:** #e11d21 (red)
- **Description:** Core functionality and essential components

### `ai`
- **Color:** #c62828 (dark red)
- **Description:** Artificial Intelligence and machine learning components

### `android`
- **Color:** #388e3c (green)
- **Description:** Android-specific functionality and components

### `accessibility`
- **Color:** #1976d2 (blue)
- **Description:** Accessibility features and senior mode functionality

### `privacy`
- **Color:** #7b1fa2 (purple)
- **Description:** Privacy and security related issues

### `documentation`
- **Color:** #f57c00 (orange)
- **Description:** Documentation tasks and improvements

### `critical`
- **Color:** #d32f2f (dark red)
- **Description:** Critical issues that require immediate attention

## Additional Recommended Labels

### `bug`
- **Color:** #e11d21 (red)
- **Description:** Something isn't working

### `enhancement`
- **Color:** #1976d2 (blue)
- **Description:** New feature or request

### `maintenance`
- **Color:** #7b1fa2 (purple)
- **Description:** Routine maintenance tasks

### `testing`
- **Color:** #f57c00 (orange)
- **Description:** Testing and validation tasks

### `wontfix`
- **Color:** #9e9e9e (gray)
- **Description:** This will not be worked on

### `duplicate`
- **Color:** #9e9e9e (gray)
- **Description:** This issue or pull request already exists

### `help wanted`
- **Color:** #0e8a16 (green)
- **Description:** Extra attention is needed

### `good first issue`
- **Color:** #0e8a16 (green)
- **Description:** Good for newcomers

## How to Create These Labels

1. Go to your GitHub repository
2. Click on "Issues" tab
3. Click on "Labels" 
4. Click "New label" for each label
5. Enter the name, color, and description as specified above
