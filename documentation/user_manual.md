# Egyptian Agent User Manual

## Getting Started

The Egyptian Agent is a voice-controlled assistant designed for Egyptian seniors and visually impaired users. It works completely hands-free using voice commands in Egyptian dialect.

## Basic Commands

### Activation
- Say "يا صاحبي" to activate the assistant
- In senior mode, say "يا كبير" to activate

### Making Calls
- "اتصل بأمي" - Call mother
- "كلم بابا" - Call father
- "رن على ماما" - Call mother
- "اتصل بـ [name]" - Call any contact

### WhatsApp Messages
- "ابعت واتساب لـ [name]" - Send WhatsApp message
- "قول لـ [name] إن [message]" - Send specific message

### Setting Alarms
- "نبهني بكرة الصبح" - Set alarm for tomorrow morning
- "انبهني بعد ساعة" - Set alarm for 1 hour from now
- "ذكرني [time]" - Set reminder for specific time

### Reading Missed Calls
- "قولي المكالمات الفايتة" - Read missed calls
- "شوفلي الفايتة" - Check missed calls

## Senior Mode Features

### Activating Senior Mode
- Say "يا كبير، شغل وضع كبار السن"
- Or press the senior mode button in the app

### Senior Mode Benefits
- Slower, clearer speech
- Simplified command set
- Double confirmation for actions
- Automatic fall detection
- Emergency assistance

### Emergency Features
- Automatic fall detection
- Triple-volume-button press for emergency
- Direct connection to emergency services
- Location sharing in emergencies

## Troubleshooting

### Assistant Not Responding
- Make sure you're saying "يا صاحبي" clearly
- Check that the app has microphone permissions
- Ensure the service is running in the background

### Call Not Working
- Verify contact exists in your phone
- Check that the app has call permissions
- Make sure you're saying the contact name clearly

### WhatsApp Not Working
- Ensure WhatsApp is installed on your device
- Check that the app has the necessary permissions
- Verify the contact has WhatsApp

## Privacy and Security

The Egyptian Agent takes your privacy seriously:
- All voice processing happens offline
- No personal data is sent to external servers
- Call logs and contacts are only accessed locally
- All data is encrypted on your device
